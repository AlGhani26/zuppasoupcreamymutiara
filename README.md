# Zuppa Soup Creamy Pondok Mutiara 🍲

Website katalog menu interaktif untuk **Zuppa Soup Creamy Pondok Mutiara**. Aplikasi web ini dibangun menggunakan teknologi modern untuk memberikan pengalaman pengguna yang cepat, responsif, dan menarik.

## 🚀 Teknologi yang Digunakan

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Library UI**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Ikon**: [Lucide React](https://lucide.dev/)
- **Bahasa**: TypeScript

## 📁 Struktur Proyek

- `src/app`: Berisi halaman utama dan konfigurasi routing menggunakan Next.js App Router.
- `src/components`: Komponen UI yang dapat digunakan kembali untuk membangun antarmuka (contoh: `MenuListing`, `BurgerSection`, `SidesPastaSection`, `MenuFavorites`).
- `src/assets`: Penyimpanan untuk berbagai aset statis seperti gambar.

## 🛠️ Panduan Memulai (Getting Started)

### Prasyarat

Pastikan Anda telah menginstal [Node.js](https://nodejs.org/) (disarankan menggunakan versi 20 atau yang lebih baru) di sistem Anda.

### Instalasi

1. Pastikan Anda berada di direktori proyek:
   ```bash
   cd zuppasoupcreamymutiara
   ```

2. Instal semua dependensi yang diperlukan:
   ```bash
   npm install
   ```
   *Atau Anda dapat menggunakan `yarn`, `pnpm`, atau `bun` sesuai preferensi.*

### Menjalankan Server Pengembangan

Untuk menjalankan aplikasi di lingkungan pengembangan (development):

```bash
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser untuk melihat aplikasi. Halaman akan diperbarui secara otomatis saat Anda melakukan perubahan pada kode.

## 📜 Skrip yang Tersedia

Di dalam direktori proyek, Anda dapat menjalankan perintah berikut:

- `npm run dev`: Menjalankan aplikasi dalam mode pengembangan.
- `npm run build`: Membangun (compile) aplikasi yang dioptimalkan untuk siap rilis (produksi).
- `npm run start`: Menjalankan aplikasi dari hasil build produksi.
- `npm run lint`: Menjalankan ESLint untuk mencari dan memperbaiki masalah penulisan pada kode.
